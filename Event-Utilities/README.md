# Project Utilities Event

A collection of utility scripts and web-based tools for event management, file organization, and image highlighting.

## Directory Structure

### 1. 3-3 Highlight
A web-based image gallery and highlighting tool.
- **Files:** `main.html`, `main2.html`, `main3.html`
- **Assets:** Contains various `.jpg` and `.png` images.

### 2. File Tools
A suite of Python and PowerShell scripts for file management and automation.
- **Python Scripts:**
    - `converter.py`: Likely for file format conversion.
    - `xl_merger.py`: Merges Excel files.
    - `gem_version.py`: Version management.
    - `generate_tree.py`: Generates directory trees.
- **PowerShell Scripts:**
    - `organizer_(전기고,행운권).ps1`: Organizes files for specific events.
    - `compress_video_install.ps1`: Video compression setup.
    - `symbolic_link_maker.ps1`: Helper for creating symbolic links.
- **NEIS Tools:**
    - `neis_helper web.py`: Helper script for NEIS web interactions.

### 3. Lottery Event
A simple tool for lucky draws or lottery events.
- **Files:** `pick.html`, `행운권뽑기.csv`

## Setup & Usage

### Prerequisites
- **Python 3.x**
- **PowerShell** (for Windows scripts)
- **Web Browser** (for HTML tools)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rjegjin/Project_Utilities_Event.git
   ```
2. Navigate to the directory:
   ```bash
   cd Project_Utilities_Event
   ```
